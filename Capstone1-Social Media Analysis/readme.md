## ğŸ§  Key Insights

* **Likes** are the strongest driver of engagement rate (high linear correlation)
* **Beauty, Photography, and Lifestyle** are the top-performing categories
* **Explore** traffic generates the most engagement, but traffic source overall has minimal impact
* **Video content** slightly outperforms photos, but media type is not a major driver
* Engagement patterns show **event-based spikes**, not continuous trends
* Posting during historically low-engagement weeks reduces effectiveness
